# Changelog
All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-10

### Added
- **Reservation Feature**: Comprehensive room reservation system with full CRUD (Create, Read, Update, Delete) capabilities.
- **Room Information Feature**: Real-time room availability status based on selected dates.
- **Reservation History Feature**: Dedicated view for historical reservation data and archived bookings.
- **Search System**: Efficient server-side filtering for active reservations to improve data retrieval.
- **Security & Data Integrity**: Implemented DTOs (Data Transfer Objects) to secure API contracts and prevent over-posting.
